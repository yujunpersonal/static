隐私政策

最后更新：2026年4月27日

本隐私政策说明当您使用通过 Apple App Store 分发的我们的 App 时， 我们如何收集、使用和保护信息。

1. 我们收集的数据及其使用方式

根据您所使用的功能，我们的不同 App 会收集不同类别的数据。 以下类别描述的是我们 App 涵盖的最大范围； 单个 App 仅会收集其功能所需的数据。

  • 音频录音。仅在您主动在 App 内开始录音或语音交互时， 由您设备的麦克风进行采集。用于将您所说的内容转写为文字， 并在适用时将您的语音请求发送给 AI 服务，以便 App 给出回应。
  • 转写文本和文字提示。您语音输入的文本版本，以及 您输入到提示框或聊天框中的任何文字。用于生成 App 的回应， 并在 App 支持对话历史的情况下，于您的设备上向您显示过往交互。
  • 图片和照片。仅限您在 App 内明确选择的图片 （例如选择头像时）。仅用于您选择该图片时所对应的应用内目的。
  • 购买和订阅信息。当您完成购买时，由 Apple 的 StoreKit 提供。 我们仅接收交易标识符和订阅状态；我们不接收您的支付详细信息。
  • 本地存储的偏好设置。App 设置、语言选择，以及（在适用时） 您选择提供给第三方 AI 服务的 API 密钥。这些数据存储在您的设备上—— 偏好设置存储在系统默认存储中，API 密钥存储在 iOS 钥匙串中。 它们不会传输给我们。
  • 诊断信息。崩溃日志和聚合的、不可识别个人身份的性能数据， 仅在您已在 iOS 设置 → 隐私与安全 → 分析与改进中启用此项时才会收集。

我们不会收集您的姓名、电子邮件地址、通讯录、精确位置或用于广告的设备标识符。 我们不会出售您的数据，也不会使用您的数据训练任何机器学习模型。

2. AI 功能与第三方 AI 服务提供商

我们的部分 App 包含由第三方人工智能服务驱动的功能。当 App 提供此类功能时， 在您的任何数据被发送之前，App 内的功能说明和同意提示会告知您 所使用的服务提供商。

根据具体的 App 和您选择的服务提供商，以下数据可能从您的设备直接传输到 一个或多个此类服务提供商：

  • 您在 App 中创建的音频录音
  • 您音频的转写文本
  • 您输入的文字提示
  • 为 AI 提供回复上下文所需的对话历史

我们可能向其传输此类数据的第三方 AI 服务包括：

  • OpenAI, L.L.C. — 语音转文字、语言模型和文字转语音 API。 请参阅 OpenAI 隐私政策： https://openai.com/policies/privacy-policy， 以及其 API 数据使用政策： https://openai.com/policies/api-data-usage-policies。
  • Anthropic, PBC — 语言模型 API（Claude）。请参阅 Anthropic 隐私政策： https://www.anthropic.com/legal/privacy。
  • Google LLC — Gemini 语言模型 API。请参阅 Google 隐私政策： https://policies.google.com/privacy， 以及 Google AI 条款： https://ai.google.dev/gemini-api/terms。

若 App 支持“自带密钥”模式——即由您自行提供 AI 服务提供商的 API 密钥—— 则您的数据将以您在该提供商处的账户名义，从您的设备直接发送给该提供商， 您的使用受您与该提供商之间协议的约束。

我们已审阅上述每家服务提供商所公布的政策，并依据其商业条款使用其 API， 要求其为您的数据提供与本政策所述相当的保护措施。 根据上述链接的政策，这些服务提供商默认不会使用 API 输入或输出来训练其模型。

3. 您的选择以及如何删除数据

  • 您可以随时通过在 App 设置中移除 API 密钥（在适用情况下）， 或在首次出现同意提示时拒绝同意，以阻止 App 使用 AI 功能。
  • 您可以通过 App 设置删除存储在您设备上的所有对话历史和录音。
  • 您可以随时删除 App，这将移除所有本地存储的数据， 包括钥匙串中保存的任何 API 密钥。
  • 如需请求删除我们持有的关于您的任何数据，请通过下方地址联系我们。

4. 应用内购买与订阅

付款由 Apple 处理，而非由我们直接处理。我们可能接收提供付费功能和客户支持所需的 订阅状态和交易元数据。

5. 儿童

我们的 App 面向一般受众，并不针对 13 岁以下儿童（或您所在司法管辖区的等同年龄）。 我们不会有意收集 13 岁以下儿童的个人信息。 若您认为有儿童向我们提供了个人信息，请联系我们，我们将予以删除。

6. 数据保留

我们会在 App 安装在您设备上的期间保留本地存储的 App 数据。 与您的 Apple ID 相关联的购买记录会根据 Apple 及适用税法的要求予以保留。 我们不会为您维护服务器端的账户或资料。

7. 安全

您提供的 API 密钥存储在 iOS 钥匙串中，由操作系统进行加密。 向第三方 AI 服务提供商发出的网络请求均通过 HTTPS 进行。

8. 本政策的变更

我们可能不时更新本政策。上方的“最后更新”日期反映最近一次变更。 重大变更将在下一次启动时通过 App 内的同意流程予以体现。

9. 联系我们

如有隐私或数据请求，请发送邮件至 yujun.china@gmail.com。 您也可访问我们的支持页面，或查看我们的 用户隐私声明、 EULA 和使用条款。
