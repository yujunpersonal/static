プライバシーポリシー

最終更新日: 2026年4月27日

本プライバシーポリシーは、Apple App Store を通じて配信される当社のアプリをご利用いただく際に、 当社がどのように情報を収集、使用、保護するかについて説明します。

1. 当社が収集するデータとその使用方法

当社の各アプリは、ご利用の機能に応じて異なるカテゴリのデータを収集します。 以下のカテゴリは当社のアプリ全体での最大範囲を示すもので、個々のアプリは その機能に必要な情報のみを収集します。

  • 音声録音。アプリ内で録音または音声操作を実際に開始したときにのみ、 お使いのデバイスのマイクで取得します。発話内容を文字起こしし、必要に応じて、 アプリが応答できるよう音声リクエストを AI サービスに送信するために使用します。
  • 文字起こしおよびテキストプロンプト。音声入力のテキスト版、および プロンプトやチャットフィールドに入力したテキスト。アプリの応答を生成するため、 また、アプリが会話履歴に対応している場合は、過去のやり取りをデバイス上に表示するために使用します。
  • 画像と写真。アプリ内で選択した特定の画像のみ(例: アバターを選ぶときなど)。 選択したアプリ内目的のためにのみ使用されます。
  • 購入およびサブスクリプション情報。購入時に Apple の StoreKit から提供されます。 取引識別子とサブスクリプションのステータスを受け取りますが、お支払い情報は受け取りません。
  • ローカルに保存される設定。アプリ設定、言語の選択、および(該当する場合) 第三者の AI サービス用にご自身で提供する API キー。これらはお使いのデバイスに保存されます。 設定はシステムのデフォルトストアに、API キーは iOS キーチェーンに保存されます。 これらは当社に送信されません。
  • 診断情報。クラッシュログおよび個人を特定しない集計されたパフォーマンスデータで、 iOS の「設定」→「プライバシーとセキュリティ」→「解析と改善」で有効にしている場合のみ収集されます。

当社は、お客様の氏名、メールアドレス、連絡先リスト、正確な位置情報、または広告に使用される デバイス識別子を収集しません。当社はお客様のデータを販売せず、機械学習モデルの学習に お客様のデータを使用することもありません。

2. AI 機能および第三者の AI プロバイダー

当社の一部のアプリには、第三者の人工知能サービスを利用した機能が含まれています。 アプリがそうした機能を提供する場合、データが送信される前に、アプリ内の説明と同意プロンプトで どのプロバイダーが使用されているかをお知らせします。

アプリと選択したプロバイダーに応じて、以下のデータがお使いのデバイスから 1 社以上の プロバイダーに直接送信される場合があります:

  • アプリ内で作成した音声録音
  • 音声の文字起こし
  • 入力したテキストプロンプト
  • AI が応答するための文脈として必要な会話履歴

これらのデータを送信する可能性がある第三者の AI サービスは以下のとおりです:

  • OpenAI, L.L.C. — 音声テキスト変換、言語モデル、およびテキスト読み上げ API。 OpenAI のプライバシーポリシーは https://openai.com/policies/privacy-policy を、API データ利用ポリシーは https://openai.com/policies/api-data-usage-policies をご参照ください。
  • Anthropic, PBC — 言語モデル API(Claude)。Anthropic のプライバシーポリシーは https://www.anthropic.com/legal/privacy をご参照ください。
  • Google LLC — Gemini 言語モデル API。Google のプライバシーポリシーは https://policies.google.com/privacy を、Google AI の利用規約は https://ai.google.dev/gemini-api/terms をご参照ください。

アプリが「Bring Your Own Key(独自の API キーを使用する)」モデルに対応している場合、 つまり AI プロバイダーの API キーをご自身で提供する場合、お客様のデータはお使いのデバイスから そのプロバイダーへ、お客様自身のアカウントのもとで直接送信され、その利用はお客様と当該 プロバイダーとの契約に従います。

当社は各プロバイダーが公開するポリシーを確認しており、その商用条件のもとで API を利用することを通じて、 本ポリシーに記載された内容と同等の保護をお客様のデータに対して提供することを各プロバイダーに 要求しています。上記にリンクしたポリシーに従い、これらのプロバイダーは API の入出力を モデルの学習にデフォルトで使用しません。

3. お客様の選択肢とデータの削除方法

  • アプリの設定から API キーを削除する(該当する場合)、または初回表示時に同意プロンプトを拒否することで、 いつでもアプリによる AI 機能の使用を停止できます。
  • アプリの設定から、デバイスに保存されているすべての会話履歴と録音を削除できます。
  • アプリはいつでも削除でき、これによりキーチェーンに保存されている API キーを含む、 ローカルに保存されたすべてのデータが削除されます。
  • 当社が保持しているお客様に関するデータの削除を希望される場合は、以下のアドレスまでご連絡ください。

4. アプリ内課金とサブスクリプション

お支払いは当社ではなく Apple が処理します。当社は有料機能の提供および カスタマーサポートに必要なサブスクリプションのステータスや取引メタデータを受け取る場合があります。

5. お子様について

当社のアプリは一般向けであり、13 歳未満(またはお住まいの法域における同等の年齢)のお子様を 対象としていません。当社は 13 歳未満のお子様から個人情報を意図的に収集することはありません。 お子様が当社に個人情報を提供したと思われる場合は、ご連絡いただければ削除いたします。

6. データの保持

ローカルに保存されたアプリデータは、アプリがお使いのデバイスにインストールされている限り保持されます。 Apple ID に関連付けられた購入記録は、Apple および適用される税法の要件に従って保持されます。 当社はお客様向けのサーバー側アカウントやプロフィールを保持していません。

7. セキュリティ

お客様が提供する API キーは、オペレーティングシステムによって暗号化される iOS キーチェーンに保存されます。 第三者の AI プロバイダーへのネットワークリクエストは HTTPS を介して行われます。

8. 本ポリシーの変更

当社は本ポリシーを随時更新する場合があります。上記の「最終更新日」は、最新の変更を反映しています。 重要な変更がある場合は、次回起動時にアプリ内の同意フローに反映されます。
