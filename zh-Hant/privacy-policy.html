隱私權政策

最後更新：2026年4月27日

本隱私權政策說明當您使用透過 Apple App Store 散布的我們的 App 時， 我們如何收集、使用並保護資訊。

1. 我們收集的資料及其使用方式

視您所使用的功能而定，我們的不同 App 會收集不同類別的資料。 下列類別描述的是我們 App 所涵蓋的最大範圍； 個別 App 僅會收集其功能所需的資料。

  • 音訊錄音。僅在您於 App 內主動開始錄音或語音互動時， 由您裝置的麥克風進行擷取。用於將您所說的內容轉寫為文字， 並在適用時將您的語音請求傳送至 AI 服務，以便 App 做出回應。
  • 轉寫文字與文字提示。您語音輸入的文字版本，以及 您在提示欄或聊天欄中輸入的任何文字。用於產生 App 的回應， 並在 App 支援對話歷史的情況下，於您的裝置上向您顯示過往的互動。
  • 影像與相片。僅限您在 App 內明確選擇的影像 （例如選擇個人頭像時）。僅用於您選擇該影像時所對應的應用內用途。
  • 購買與訂閱資訊。當您完成購買時，由 Apple 的 StoreKit 提供。 我們僅會收到交易識別碼及訂閱狀態；我們不會收到您的付款細節。
  • 本機儲存的偏好設定。App 設定、語言選擇，以及（在適用時） 您選擇提供給第三方 AI 服務的 API 金鑰。這些資料儲存於您的裝置上—— 偏好設定儲存於系統預設值儲存區中，API 金鑰儲存於 iOS Keychain 中。 它們不會傳送給我們。
  • 診斷資訊。當機紀錄及彙總的、無法識別個人身分的效能資料， 僅在您已於 iOS 設定 → 隱私權與安全性 → 分析與改進中啟用此項目時才會收集。

我們不會收集您的姓名、電子郵件地址、通訊錄、精確位置， 或用於廣告的裝置識別碼。我們不會出售您的資料， 也不會使用您的資料訓練任何機器學習模型。

2. AI 功能與第三方 AI 服務提供者

我們的部分 App 包含由第三方人工智慧服務所驅動的功能。當 App 提供此類功能時， 在您的任何資料被傳送之前，App 內的功能說明與同意提示將會告知您 所使用的服務提供者。

依您所使用的 App 與所選擇的服務提供者而定，下列資料可能會從您的裝置 直接傳輸至一個或多個此類服務提供者：

  • 您於 App 中建立的音訊錄音
  • 您音訊的轉寫文字
  • 您輸入的文字提示
  • 為 AI 提供回覆所需上下文的對話歷史

我們可能將此類資料傳輸至的第三方 AI 服務包括：

  • OpenAI, L.L.C. — 語音轉文字、語言模型與文字轉語音 API。 請參閱 OpenAI 隱私權政策： https://openai.com/policies/privacy-policy， 以及其 API 資料使用政策： https://openai.com/policies/api-data-usage-policies。
  • Anthropic, PBC — 語言模型 API（Claude）。請參閱 Anthropic 隱私權政策： https://www.anthropic.com/legal/privacy。
  • Google LLC — Gemini 語言模型 API。請參閱 Google 隱私權政策： https://policies.google.com/privacy， 以及 Google AI 條款： https://ai.google.dev/gemini-api/terms。

當 App 支援「自帶金鑰」模式——亦即由您自行提供 AI 服務提供者的 API 金鑰—— 您的資料會以您在該服務提供者處的帳號名義，從您的裝置直接傳送至該服務提供者， 您的使用將受您與該服務提供者之間協議的規範。

我們已審閱上述每位服務提供者所公布的政策，並依據其商業條款使用其 API， 要求其為您的資料提供與本政策所述相當的保護措施。 依據上方連結的政策，這些服務提供者預設不會使用 API 輸入或輸出來訓練其模型。

3. 您的選擇以及如何刪除資料

  • 您可隨時於 App 設定中移除 API 金鑰（在適用情況下）， 或在首次出現同意提示時拒絕同意，以停止 App 使用 AI 功能。
  • 您可透過 App 設定刪除儲存於您裝置上的所有對話歷史與錄音。
  • 您可隨時刪除 App，這將移除所有本機儲存的資料， 包括 Keychain 中儲存的任何 API 金鑰。
  • 如需請求刪除我們所持有有關您的任何資料，請透過下方地址與我們聯絡。

4. 應用程式內購買與訂閱

付款由 Apple 處理，並非由我們直接處理。我們可能會收到提供付費功能與客戶支援所需的 訂閱狀態與交易中繼資料。

5. 兒童

我們的 App 面向一般大眾，並不針對 13 歲以下兒童（或您所在司法管轄區的同等年齡）。 我們不會有意收集 13 歲以下兒童的個人資訊。 若您認為有兒童向我們提供了個人資訊，請與我們聯絡，我們將予以刪除。

6. 資料保留

我們會在 App 安裝於您裝置上的期間保留本機儲存的 App 資料。 與您 Apple ID 相關聯的購買紀錄將依 Apple 及適用稅法之要求予以保留。 我們不會為您維護伺服器端的帳號或個人檔案。

7. 安全性

您所提供的 API 金鑰會儲存於 iOS Keychain 中，並由作業系統加密。 對第三方 AI 服務提供者的網路請求均透過 HTTPS 進行。

8. 本政策的變更

我們可能不時更新本政策。上方的「最後更新」日期反映最近一次的變更。 重大變更將於下次啟動時透過 App 內的同意流程加以呈現。

9. 聯絡我們

如有隱私權或資料請求，請來信 yujun.china@gmail.com。 您也可以造訪我們的支援頁面，或查看我們的 使用者隱私權聲明、 EULA 與使用條款。
