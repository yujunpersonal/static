Πολιτική Απορρήτου

Τελευταία ενημέρωση: 27 Απριλίου 2026

Η παρούσα Πολιτική Απορρήτου εξηγεί πώς συλλέγουμε, χρησιμοποιούμε και προστατεύουμε πληροφορίες όταν χρησιμοποιείτε τις εφαρμογές μας που διανέμονται μέσω του Apple App Store.

1. Δεδομένα που συλλέγουμε και πώς τα χρησιμοποιούμε

Διαφορετικές εφαρμογές μας συλλέγουν διαφορετικές κατηγορίες δεδομένων, ανάλογα με τις λειτουργίες που χρησιμοποιείτε. Οι παρακάτω κατηγορίες περιγράφουν το μέγιστο εύρος σε όλες τις εφαρμογές μας· κάθε επιμέρους εφαρμογή συλλέγει μόνο όσα απαιτούνται για τις λειτουργίες της.

  • Ηχογραφήσεις. Καταγράφονται από το μικρόφωνο της συσκευής σας μόνο όταν ξεκινάτε ενεργά μια εγγραφή ή φωνητική αλληλεπίδραση μέσα στην εφαρμογή. Χρησιμοποιούνται για την απομαγνητοφώνηση όσων είπατε και, όπου ισχύει, για την αποστολή του προφορικού σας αιτήματος σε μια υπηρεσία τεχνητής νοημοσύνης ώστε να μπορέσει η εφαρμογή να απαντήσει.
  • Απομαγνητοφωνήσεις και προτροπές κειμένου. Η σε κείμενο εκδοχή της φωνητικής σας εισόδου, καθώς και κάθε κείμενο που πληκτρολογείτε σε πεδίο προτροπής ή συνομιλίας. Χρησιμοποιούνται για τη δημιουργία της απάντησης της εφαρμογής και, στις εφαρμογές που υποστηρίζουν ιστορικό συνομιλιών, για την εμφάνιση παρελθουσών αλληλεπιδράσεων στη συσκευή σας.
  • Εικόνες και φωτογραφίες. Μόνο οι συγκεκριμένες εικόνες που επιλέγετε στην εφαρμογή (για παράδειγμα, όταν επιλέγετε ένα avatar). Χρησιμοποιούνται αποκλειστικά για τον σκοπό εντός της εφαρμογής για τον οποίο τις επιλέξατε.
  • Πληροφορίες αγοράς και συνδρομής. Παρέχονται από το StoreKit της Apple όταν πραγματοποιείτε μια αγορά. Λαμβάνουμε ένα αναγνωριστικό συναλλαγής και την κατάσταση συνδρομής· δεν λαμβάνουμε τα στοιχεία πληρωμής σας.
  • Τοπικά αποθηκευμένες προτιμήσεις. Ρυθμίσεις της εφαρμογής, επιλογή γλώσσας και (όπου ισχύει) το κλειδί API που επιλέγετε να παρέχετε για μια υπηρεσία τεχνητής νοημοσύνης τρίτου μέρους. Αυτά αποθηκεύονται στη συσκευή σας — οι προτιμήσεις στον αποθηκευτικό χώρο των προεπιλογών του συστήματος και τα κλειδιά API στο iOS Keychain. Δεν μεταδίδονται σε εμάς.
  • Διαγνωστικά. Καταγραφές κατάρρευσης και συγκεντρωτικά, μη ταυτοποιήσιμα δεδομένα απόδοσης, μόνο εάν το έχετε ενεργοποιήσει στις Ρυθμίσεις iOS → Απόρρητο και Ασφάλεια → Αναλύσεις και Βελτιώσεις.

Δεν συλλέγουμε το όνομά σας, τη διεύθυνση email, τη λίστα επαφών σας, την ακριβή τοποθεσία σας ή αναγνωριστικά συσκευής που χρησιμοποιούνται για διαφημιστικούς σκοπούς. Δεν πουλάμε τα δεδομένα σας και δεν τα χρησιμοποιούμε για την εκπαίδευση οποιουδήποτε μοντέλου μηχανικής μάθησης.

2. Λειτουργίες τεχνητής νοημοσύνης και πάροχοι AI τρίτων

Ορισμένες από τις εφαρμογές μας περιλαμβάνουν λειτουργίες που υποστηρίζονται από υπηρεσίες τεχνητής νοημοσύνης τρίτων. Όταν μια εφαρμογή προσφέρει τέτοια λειτουργία, η περιγραφή εντός της εφαρμογής και ένα μήνυμα συγκατάθεσης θα σας ενημερώσουν ποιος πάροχος χρησιμοποιείται πριν αποσταλεί οποιοδήποτε δεδομένο σας.

Ανάλογα με την εφαρμογή και τον πάροχο που επιλέγετε, τα ακόλουθα δεδομένα ενδέχεται να μεταδοθούν απευθείας από τη συσκευή σας σε έναν ή περισσότερους από αυτούς τους παρόχους:

  • Ηχογραφήσεις που δημιουργείτε στην εφαρμογή
  • Απομαγνητοφωνήσεις του ήχου σας
  • Προτροπές κειμένου που πληκτρολογείτε
  • Ιστορικό συνομιλίας απαραίτητο για να δοθεί στην AI το πλαίσιο για την απάντησή της

Οι υπηρεσίες AI τρίτων στις οποίες ενδέχεται να μεταδώσουμε αυτά τα δεδομένα είναι:

  • OpenAI, L.L.C. — API για ομιλία προς κείμενο, γλωσσικά μοντέλα και κείμενο προς ομιλία. Δείτε την πολιτική απορρήτου της OpenAI στο https://openai.com/policies/privacy-policy και την πολιτική χρήσης δεδομένων API της στο https://openai.com/policies/api-data-usage-policies.
  • Anthropic, PBC — API γλωσσικών μοντέλων (Claude). Δείτε την πολιτική απορρήτου της Anthropic στο https://www.anthropic.com/legal/privacy.
  • Google LLC — API του γλωσσικού μοντέλου Gemini. Δείτε την πολιτική απορρήτου της Google στο https://policies.google.com/privacy και τους όρους Google AI στο https://ai.google.dev/gemini-api/terms.

Όταν μια εφαρμογή υποστηρίζει το μοντέλο «φέρτε το δικό σας κλειδί» — δηλαδή παρέχετε εσείς το δικό σας κλειδί API για τον πάροχο AI — τα δεδομένα σας αποστέλλονται απευθείας από τη συσκευή σας σε αυτόν τον πάροχο υπό τον δικό σας λογαριασμό σε αυτόν, και η χρήση σας διέπεται από τη συμφωνία σας με τον εν λόγω πάροχο.

Έχουμε εξετάσει τις δημοσιευμένες πολιτικές κάθε ενός από αυτούς τους παρόχους και απαιτούμε, βάσει της χρήσης των API τους με τους εμπορικούς τους όρους, να παρέχουν για τα δεδομένα σας προστασία ισοδύναμη με αυτή που περιγράφεται στην παρούσα πολιτική. Σύμφωνα με τις πολιτικές που συνδέονται παραπάνω, αυτοί οι πάροχοι δεν χρησιμοποιούν εξ ορισμού τις εισόδους ή εξόδους του API για την εκπαίδευση των μοντέλων τους.

3. Οι επιλογές σας και πώς να διαγράψετε δεδομένα

  • Μπορείτε ανά πάσα στιγμή να σταματήσετε την εφαρμογή από το να χρησιμοποιεί λειτουργίες AI αφαιρώντας το κλειδί API από τις ρυθμίσεις της εφαρμογής (όπου ισχύει) ή απορρίπτοντας το μήνυμα συγκατάθεσης την πρώτη φορά που εμφανίζεται.
  • Μπορείτε να διαγράψετε όλο το ιστορικό συνομιλιών και τις ηχογραφήσεις που είναι αποθηκευμένα στη συσκευή σας μέσα από τις ρυθμίσεις της εφαρμογής.
  • Μπορείτε να διαγράψετε την εφαρμογή ανά πάσα στιγμή, κάτι που αφαιρεί όλα τα τοπικά αποθηκευμένα δεδομένα, συμπεριλαμβανομένου οποιουδήποτε αποθηκευμένου κλειδιού API στο Keychain.
  • Για να ζητήσετε τη διαγραφή οποιωνδήποτε δεδομένων διατηρούμε για εσάς, επικοινωνήστε μαζί μας στη διεύθυνση παρακάτω.

4. Αγορές εντός εφαρμογής και συνδρομές

Οι πληρωμές διεκπεραιώνονται από την Apple, όχι απευθείας από εμάς. Ενδέχεται να λαμβάνουμε την κατάσταση συνδρομής και μεταδεδομένα συναλλαγών απαραίτητα για την παροχή επί πληρωμή λειτουργιών και υποστήριξης πελατών.

5. Παιδιά

Οι εφαρμογές μας προορίζονται για ευρύ κοινό και δεν απευθύνονται σε παιδιά κάτω των 13 ετών (ή την αντίστοιχη ηλικία στη δικαιοδοσία σας). Δεν συλλέγουμε εν γνώσει μας προσωπικές πληροφορίες από παιδιά κάτω των 13 ετών. Εάν πιστεύετε ότι ένα παιδί μάς έχει παράσχει προσωπικές πληροφορίες, επικοινωνήστε μαζί μας και θα τις διαγράψουμε.

6. Διατήρηση δεδομένων

Διατηρούμε τα τοπικά αποθηκευμένα δεδομένα της εφαρμογής για όσο χρόνο η εφαρμογή είναι εγκατεστημένη στη συσκευή σας. Τα αρχεία αγορών που σχετίζονται με το Apple ID σας διατηρούνται όπως απαιτείται από την Apple και την ισχύουσα φορολογική νομοθεσία. Δεν διατηρούμε λογαριασμούς ή προφίλ στην πλευρά του διακομιστή για εσάς.

7. Ασφάλεια

Τα κλειδιά API που παρέχετε αποθηκεύονται στο iOS Keychain, το οποίο κρυπτογραφείται από το λειτουργικό σύστημα. Τα αιτήματα δικτύου προς παρόχους AI τρίτων πραγματοποιούνται μέσω HTTPS.

8. Αλλαγές στην παρούσα πολιτική

Ενδέχεται να ενημερώνουμε κατά καιρούς αυτή την πολιτική. Η ημερομηνία «Τελευταία ενημέρωση» παραπάνω αντικατοπτρίζει την πιο πρόσφατη αλλαγή. Ουσιώδεις αλλαγές θα αντικατοπτρίζονται στη ροή συγκατάθεσης εντός της εφαρμογής κατά την επόμενη εκκίνηση.
