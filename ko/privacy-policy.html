개인정보 처리방침

최종 업데이트: 2026년 4월 27일

본 개인정보 처리방침은 Apple App Store를 통해 배포되는 당사의 앱을 사용하실 때 당사가 정보를 수집, 사용, 보호하는 방법에 관해 설명합니다.

1. 당사가 수집하는 데이터 및 사용 방법

당사의 앱은 사용하시는 기능에 따라 서로 다른 카테고리의 데이터를 수집합니다. 아래 카테고리는 당사의 앱 전반에서 수집할 수 있는 최대 범위를 설명한 것이며, 개별 앱은 해당 기능에 필요한 정보만 수집합니다.

  • 오디오 녹음. 앱 내에서 녹음 또는 음성 상호작용을 적극적으로 시작했을 때만 기기의 마이크로 캡처됩니다. 발화 내용을 텍스트로 변환하고, 해당되는 경우, 앱이 응답할 수 있도록 음성 요청을 AI 서비스에 전송하는 데 사용됩니다.
  • 전사본 및 텍스트 프롬프트. 음성 입력의 텍스트 버전과 프롬프트나 채팅 필드에 입력하신 모든 텍스트입니다. 앱의 응답을 생성하는 데 사용되며, 앱이 대화 기록을 지원하는 경우 과거의 상호작용을 기기에 표시하는 데에도 사용됩니다.
  • 이미지 및 사진. 앱에서 직접 선택하신 특정 이미지만 해당됩니다(예: 아바타 선택 시). 선택하신 앱 내 목적으로만 사용됩니다.
  • 구매 및 구독 정보. 구매 시 Apple의 StoreKit에서 제공됩니다. 당사는 거래 식별자와 구독 상태를 수신하며, 결제 정보는 수신하지 않습니다.
  • 로컬에 저장되는 환경설정. 앱 설정, 언어 선택, 그리고 (해당하는 경우) 제3자 AI 서비스에 사용하기 위해 직접 제공하시는 API 키. 이러한 정보는 기기에 저장되며, 환경설정은 시스템 기본값 저장소에, API 키는 iOS 키체인에 저장됩니다. 당사로 전송되지 않습니다.
  • 진단 정보. iOS 설정 → 개인정보 보호 및 보안 → 분석 및 향상에서 활성화하신 경우에만 수집되는 충돌 로그 및 식별이 불가능한 집계 성능 데이터입니다.

당사는 귀하의 이름, 이메일 주소, 연락처 목록, 정확한 위치 또는 광고에 사용되는 기기 식별자를 수집하지 않습니다. 당사는 귀하의 데이터를 판매하지 않으며, 어떠한 머신러닝 모델의 학습에도 귀하의 데이터를 사용하지 않습니다.

2. AI 기능 및 제3자 AI 제공업체

당사의 일부 앱에는 제3자 인공지능 서비스로 구동되는 기능이 포함되어 있습니다. 앱에서 그러한 기능을 제공하는 경우, 데이터가 전송되기 전에 앱 내 설명과 동의 안내를 통해 어느 제공업체가 사용되는지 알려드립니다.

앱 및 선택하신 제공업체에 따라 다음 데이터가 귀하의 기기에서 하나 이상의 제공업체로 직접 전송될 수 있습니다:

  • 앱에서 만든 오디오 녹음
  • 오디오의 전사본
  • 입력한 텍스트 프롬프트
  • AI가 응답하는 데 필요한 컨텍스트로서의 대화 기록

당사가 이 데이터를 전송할 수 있는 제3자 AI 서비스는 다음과 같습니다:

  • OpenAI, L.L.C. — 음성-텍스트 변환, 언어 모델 및 텍스트-음성 변환 API. OpenAI의 개인정보 처리방침은 https://openai.com/policies/privacy-policy 에서, API 데이터 사용 정책은 https://openai.com/policies/api-data-usage-policies 에서 확인하실 수 있습니다.
  • Anthropic, PBC — 언어 모델 API(Claude). Anthropic의 개인정보 처리방침은 https://www.anthropic.com/legal/privacy 에서 확인하실 수 있습니다.
  • Google LLC — Gemini 언어 모델 API. Google의 개인정보처리방침은 https://policies.google.com/privacy 에서, Google AI 약관은 https://ai.google.dev/gemini-api/terms 에서 확인하실 수 있습니다.

앱이 "BYOK(자체 키 사용)" 방식을 지원하는 경우, 즉 AI 제공업체에 대한 본인의 API 키를 직접 제공하시는 경우, 귀하의 데이터는 본인의 계정으로 기기에서 해당 제공업체로 직접 전송되며, 귀하의 사용은 해당 제공업체와의 계약의 적용을 받습니다.

당사는 각 제공업체가 공개한 정책을 검토하였으며, 제공업체의 상업적 약관에 따라 그들의 API를 이용함으로써 본 방침에서 설명한 것과 동등한 수준의 데이터 보호를 제공할 것을 요구하고 있습니다. 위에 링크된 정책에 따라, 이러한 제공업체는 기본적으로 API 입력 또는 출력을 모델 학습에 사용하지 않습니다.

3. 귀하의 선택권 및 데이터 삭제 방법

  • 앱 설정에서 API 키를 제거하거나(해당하는 경우), 동의 안내가 처음 표시될 때 거부함으로써 언제든지 앱이 AI 기능을 사용하지 못하도록 할 수 있습니다.
  • 앱 설정에서 기기에 저장된 모든 대화 기록 및 녹음을 삭제할 수 있습니다.
  • 언제든지 앱을 삭제할 수 있으며, 이 경우 키체인에 저장된 API 키를 포함하여 로컬에 저장된 모든 데이터가 제거됩니다.
  • 당사가 보유한 귀하에 관한 데이터의 삭제를 요청하시려면 아래 주소로 연락해 주십시오.

4. 앱 내 구매 및 구독

결제는 당사가 직접 처리하지 않으며 Apple에서 처리합니다. 당사는 유료 기능과 고객 지원을 제공하는 데 필요한 구독 상태 및 거래 메타데이터를 받을 수 있습니다.

5. 아동

당사의 앱은 일반 사용자를 대상으로 하며, 13세 미만(또는 귀하의 관할권에서 그에 상응하는 연령)의 아동을 대상으로 하지 않습니다. 당사는 13세 미만 아동으로부터 개인정보를 의도적으로 수집하지 않습니다. 만약 아동이 당사에 개인정보를 제공한 사실이 의심되는 경우, 당사에 알려주시면 해당 정보를 삭제하겠습니다.

6. 데이터 보관

로컬에 저장된 앱 데이터는 앱이 기기에 설치되어 있는 동안 보관됩니다. 귀하의 Apple ID와 연결된 구매 기록은 Apple 및 적용 가능한 세법의 요구에 따라 보관됩니다. 당사는 귀하를 위한 서버 측 계정 또는 프로필을 유지하지 않습니다.

7. 보안

귀하가 제공하시는 API 키는 운영체제에 의해 암호화되는 iOS 키체인에 저장됩니다. 제3자 AI 제공업체로의 네트워크 요청은 HTTPS를 통해 이루어집니다.
