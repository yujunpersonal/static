นโยบายความเป็นส่วนตัว

อัปเดตล่าสุด: 27 เมษายน 2569

นโยบายความเป็นส่วนตัวฉบับนี้อธิบายวิธีที่เราเก็บรวบรวม ใช้ และปกป้องข้อมูล เมื่อคุณใช้แอปของเราที่เผยแพร่ผ่าน Apple App Store

1. ข้อมูลที่เราเก็บรวบรวมและวิธีที่เราใช้งาน

แอปต่าง ๆ ของเราจะเก็บรวบรวมข้อมูลในประเภทที่แตกต่างกัน ขึ้นอยู่กับฟีเจอร์ที่คุณใช้งาน ประเภทข้อมูลด้านล่างนี้อธิบายขอบเขตสูงสุดของแอปทั้งหมดของเรา โดยแอปแต่ละตัวจะเก็บเฉพาะข้อมูลที่จำเป็นสำหรับฟีเจอร์ของแอปนั้นเท่านั้น

  • การบันทึกเสียง ถูกบันทึกโดยไมโครโฟนของอุปกรณ์ของคุณ เฉพาะเมื่อคุณเริ่มการบันทึกหรือการโต้ตอบด้วยเสียงในแอปด้วยตนเองเท่านั้น ใช้เพื่อถอดเสียงที่คุณพูดเป็นข้อความ และเมื่อสามารถใช้ได้ จะส่งคำขอทางเสียงของคุณไปยังบริการ AI เพื่อให้แอปสามารถตอบกลับได้
  • ข้อความถอดเสียงและข้อความพรอมต์ ข้อความที่ได้จากการพูดของคุณ รวมถึงข้อความที่คุณพิมพ์ในช่องพรอมต์หรือช่องแชต ใช้เพื่อสร้างคำตอบของแอป และในกรณีที่แอปรองรับประวัติการสนทนา จะใช้แสดงการโต้ตอบที่ผ่านมาบนอุปกรณ์ของคุณ
  • รูปภาพและภาพถ่าย เฉพาะรูปภาพที่คุณเลือกในแอปเท่านั้น (เช่น เมื่อคุณเลือกอวาตาร์) ใช้เพียงเพื่อวัตถุประสงค์ภายในแอปตามที่คุณเลือกเท่านั้น
  • ข้อมูลการซื้อและการสมัครสมาชิก ส่งมาให้โดย StoreKit ของ Apple เมื่อคุณทำการซื้อ เราจะได้รับเฉพาะรหัสอ้างอิงธุรกรรมและสถานะการสมัครสมาชิกเท่านั้น เราไม่ได้รับรายละเอียดการชำระเงินของคุณ
  • การตั้งค่าที่เก็บไว้ในเครื่อง การตั้งค่าแอป การเลือกภาษา และ (เมื่อสามารถใช้ได้) คีย์ API ที่คุณเลือกที่จะกรอกให้บริการ AI ของบุคคลภายนอก ข้อมูลเหล่านี้ถูกเก็บไว้ในอุปกรณ์ของคุณ — การตั้งค่าจะอยู่ในที่จัดเก็บค่าเริ่มต้นของระบบ และคีย์ API จะอยู่ใน iOS Keychain ข้อมูลเหล่านี้ไม่ถูกส่งมาให้เรา
  • ข้อมูลวินิจฉัย บันทึกการขัดข้องและข้อมูลประสิทธิภาพแบบรวมที่ไม่สามารถระบุตัวตนได้ เฉพาะเมื่อคุณได้เปิดใช้งานในการตั้งค่า iOS → ความเป็นส่วนตัวและความปลอดภัย → การวิเคราะห์และการปรับปรุง เท่านั้น

เราไม่เก็บรวบรวมชื่อ ที่อยู่อีเมล รายชื่อผู้ติดต่อ ตำแหน่งที่ตั้งที่แม่นยำ หรือรหัสประจำตัวอุปกรณ์ที่ใช้สำหรับการโฆษณาของคุณ เราไม่ขายข้อมูลของคุณ และเราไม่ใช้ข้อมูลของคุณในการฝึกโมเดลแมชชีนเลิร์นนิงใด ๆ

2. ฟีเจอร์ AI และผู้ให้บริการ AI ของบุคคลภายนอก

แอปบางแอปของเรามีฟีเจอร์ที่ขับเคลื่อนด้วยบริการปัญญาประดิษฐ์ของบุคคลภายนอก เมื่อแอปมีฟีเจอร์ดังกล่าว คำอธิบายในแอปและคำถามเพื่อขอความยินยอม จะแจ้งให้คุณทราบว่ากำลังใช้ผู้ให้บริการรายใด ก่อนที่ข้อมูลของคุณจะถูกส่งออกไป

ขึ้นอยู่กับแอปและผู้ให้บริการที่คุณเลือก ข้อมูลต่อไปนี้อาจถูกส่งโดยตรงจากอุปกรณ์ของคุณ ไปยังผู้ให้บริการเหล่านี้รายใดรายหนึ่งหรือหลายราย:

  • การบันทึกเสียงที่คุณสร้างขึ้นในแอป
  • ข้อความถอดเสียงจากเสียงของคุณ
  • ข้อความพรอมต์ที่คุณพิมพ์
  • ประวัติการสนทนาที่จำเป็นต่อการให้บริบทแก่ AI ในการตอบกลับ

บริการ AI ของบุคคลภายนอกที่เราอาจส่งข้อมูลนี้ให้ ได้แก่:

  • OpenAI, L.L.C. — API สำหรับการแปลงเสียงเป็นข้อความ โมเดลภาษา และการแปลงข้อความเป็นเสียง ดูนโยบายความเป็นส่วนตัวของ OpenAI ที่ https://openai.com/policies/privacy-policy และนโยบายการใช้ข้อมูล API ของพวกเขาที่ https://openai.com/policies/api-data-usage-policies
  • Anthropic, PBC — API โมเดลภาษา (Claude) ดูนโยบายความเป็นส่วนตัวของ Anthropic ที่ https://www.anthropic.com/legal/privacy
  • Google LLC — API โมเดลภาษา Gemini ดูนโยบายความเป็นส่วนตัวของ Google ที่ https://policies.google.com/privacy และข้อกำหนดของ Google AI ที่ https://ai.google.dev/gemini-api/terms

เมื่อแอปรองรับโมเดลแบบ "นำคีย์ของคุณเอง" — หมายความว่าคุณเป็นผู้ให้คีย์ API ของผู้ให้บริการ AI ด้วยตนเอง — ข้อมูลของคุณจะถูกส่งจากอุปกรณ์ของคุณโดยตรง ไปยังผู้ให้บริการนั้นภายใต้บัญชีของคุณกับผู้ให้บริการรายนั้น และการใช้งานของคุณจะอยู่ภายใต้ข้อตกลงระหว่างคุณกับผู้ให้บริการรายนั้น

เราได้ตรวจสอบนโยบายที่เผยแพร่ของผู้ให้บริการแต่ละรายเหล่านี้แล้ว และจากการใช้งาน API ของพวกเขาภายใต้ข้อกำหนดเชิงพาณิชย์ เรากำหนดให้พวกเขามอบการคุ้มครองข้อมูลของคุณ ในระดับที่เทียบเท่ากับที่อธิบายไว้ในนโยบายฉบับนี้ ตามนโยบายที่ลิงก์ไว้ด้านบน ผู้ให้บริการเหล่านี้ จะไม่ใช้ข้อมูลที่ป้อนเข้าหรือผลลัพธ์ของ API มาฝึกโมเดลของพวกเขาตามค่าเริ่มต้น

3. ทางเลือกของคุณและวิธีลบข้อมูล

  • คุณสามารถหยุดไม่ให้แอปใช้ฟีเจอร์ AI ได้ตลอดเวลา โดยลบคีย์ API ออกจากการตั้งค่าของแอป (เมื่อสามารถใช้ได้) หรือโดยปฏิเสธคำถามขอความยินยอมเมื่อปรากฏขึ้นเป็นครั้งแรก
  • คุณสามารถลบประวัติการสนทนาและการบันทึกเสียงทั้งหมดที่จัดเก็บไว้บนอุปกรณ์ของคุณ ได้จากภายในการตั้งค่าของแอป
  • คุณสามารถลบแอปได้ตลอดเวลา ซึ่งจะลบข้อมูลที่จัดเก็บไว้ในเครื่องทั้งหมด รวมถึงคีย์ API ที่บันทึกไว้ใน Keychain ด้วย
  • หากต้องการขอให้ลบข้อมูลใด ๆ ที่เรามีเกี่ยวกับคุณ โปรดติดต่อเราตามที่อยู่ด้านล่าง

4. การซื้อในแอปและการสมัครสมาชิก

การชำระเงินดำเนินการโดย Apple ไม่ใช่โดยเราโดยตรง เราอาจได้รับสถานะการสมัครสมาชิกและข้อมูลเมตาของธุรกรรม ที่จำเป็นต่อการให้บริการฟีเจอร์แบบเสียค่าใช้จ่ายและการสนับสนุนลูกค้า

5. เด็ก

แอปของเรามีไว้สำหรับผู้ใช้ทั่วไป และไม่ได้มุ่งเป้าไปที่เด็กอายุต่ำกว่า 13 ปี (หรืออายุที่เทียบเท่าตามเขตอำนาจศาลของคุณ) เราไม่ได้เก็บรวบรวมข้อมูลส่วนบุคคลจากเด็กอายุต่ำกว่า 13 ปีโดยรู้เท่าทัน หากคุณเชื่อว่ามีเด็กให้ข้อมูลส่วนบุคคลแก่เรา โปรดติดต่อเรา และเราจะลบข้อมูลนั้น

6. การเก็บรักษาข้อมูล

เราจะเก็บรักษาข้อมูลแอปที่จัดเก็บไว้ในเครื่องตราบเท่าที่แอปยังติดตั้งอยู่บนอุปกรณ์ของคุณ บันทึกการซื้อที่เชื่อมโยงกับ Apple ID ของคุณจะถูกเก็บรักษาไว้ตามที่ Apple และกฎหมายภาษีที่เกี่ยวข้องกำหนด เราไม่ได้ดูแลรักษาบัญชีหรือโปรไฟล์ฝั่งเซิร์ฟเวอร์สำหรับคุณ

7. ความปลอดภัย

คีย์ API ที่คุณให้ไว้จะถูกจัดเก็บใน iOS Keychain ซึ่งเข้ารหัสโดยระบบปฏิบัติการ คำขอเครือข่ายไปยังผู้ให้บริการ AI ของบุคคลภายนอกจะดำเนินการผ่าน HTTPS

8. การเปลี่ยนแปลงนโยบายฉบับนี้

เราอาจอัปเดตนโยบายฉบับนี้เป็นครั้งคราว วันที่ "อัปเดตล่าสุด" ด้านบน สะท้อนถึงการเปลี่ยนแปลงล่าสุด การเปลี่ยนแปลงที่มีนัยสำคัญจะปรากฏ ในขั้นตอนการขอความยินยอมในแอปเมื่อเปิดใช้งานครั้งถัดไป
